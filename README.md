# AI Chatbot

An interactive AI Chatbot developed using Python, Flask, HTML, CSS, and JavaScript.

## Features

- Interactive chatbot interface
- User-friendly UI
- Flask backend
- Responsive design
- Real-time user interaction

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript

## Project Structure

```
CodeAlpha_Chatbot/
│── app.py
│── requirements.txt
│── README.md
├── static/
│   ├── style.css
│   └── script.js
└── templates/
    └── index.html
```

## Installation

1. Clone the repository

```bash
git clone https://github.com/Sanikakumbhar1803/CodeAlpha_Chatbot.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the project

```bash
python app.py
```
